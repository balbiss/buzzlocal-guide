# JR Inovações

## Sobre o Projeto

Este projeto foi desenvolvido para a JR Inovações, focando em eventos e palestras que transformam conhecimento em resultados reais.

## Como rodar o projeto localmente

O único requisito é ter o Node.js e npm instalados.

Siga estes passos:

```sh
# Passo 1: Instale as dependências necessárias.
npm i

# Passo 2: Inicie o servidor de desenvolvimento.
npm run dev
```

## Tecnologias utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

O projeto pode ser buildado para produção usando:

```sh
npm run build
```

Os arquivos gerados estarão na pasta `dist`.
